# Descripción
Este proyecto se creó con la intención de hacer un sistema conceptual el cual se puedan realizar simulaciones de como funcionaria un sistema de votación digital teniendo a quien va destinado, porque se hizo y las herramientas informáticas.
## Funcionalidades
-Se pueden crear usuarios, candidatos y administradores
-Tiene registro de cambios de contraseñas y votaciones
-Se puede gestionar los distritos, departamentos, provincias, cargos, organizaciones y sus roles internos.
-Se puede crear votaciones según el tipo entre ellos puede ser: Distrital, provincial, departamental, institucional, partido (Elecciones internas para votar por sus lideres, vicepresidentes, secretarios, etc.) y referéndums.
# Archivos
Se necesitará clonar los dos repositorios desde GitHub para el funcionamiento del proyecto, se puede clonar con el siguiente código:
```
git clone https://github.com/JK-collab-hash/Frontend-G5.git
git clone https://github.com/JK-collab-hash/Backend-G5.git
```
### Instrucciones
Una vez clonado los repositorios se debe utilizar el Query que esta en la carpeta de db del Backend una vez que este 
```
# Backend (http://localhost:3000)
cd "Proyeco Integrador - Backend"
node app.js
# Frontend (http://localhost:3001)
cd "Proyeco Integrador - Frontend"
node index.js
```
